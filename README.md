# Directions
write a function that takes an input positive integer number N and
returns a NxN spiral Matrix.

# examples

## for N=2
[
  [1, 2],
  [4, 3]
]

## for N=3
[
  [1,2,3],
  [8,9,4],
  [7,6,5]
]

## for N=4
[
  [1,2,3,4],
  [12,13,14,5],
  [11,16,15,6],
  [10,9,8,7]
]


An NxN matrix where the numbers increment in a spiral.
